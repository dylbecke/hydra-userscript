# Hydra Userscript

Dual-headed load tracker - Inbound (red) and Outbound (blue) on SSP Dock pages.

## Install

1. Install Tampermonkey in your browser
2. Click: https://raw.githubusercontent.com/dylbecke/hydra-userscript/main/Hydra.user.js
3. Click Install in the Tampermonkey dialog
4. Done - updates are automatic from now on

## Auto-Updates

Once installed, Tampermonkey checks for updates periodically.
When a new version is pushed to this repo, all users get it automatically.

Force update: Tampermonkey Dashboard > Utilities > Check for userscript updates

## For Developers (Dylan / Emanuil)

1. Make changes to Hydra.user.js
2. Bump @version in BOTH Hydra.user.js and Hydra.meta.js
3. Commit and push to main
4. Users receive the update within 24 hours
